## Deploy to GitHub Pages

1. Go to **Settings > Pages**
2. Under **Source**, select **Deploy from a branch**
3. Choose `main` branch and `/ (root)` folder
4. Click **Save**

## Before Going Live

Replace the placeholder `#` links in each page's footer with your actual URLs for:
- Live Dashboard
- GitHub
- LinkedIn

## Tech

- Pure HTML + CSS + vanilla JS — no frameworks, no build step
- Google Fonts: DM Serif Display + DM Sans
- Dark mode with `localStorage` persistence
- Fully responsive (mobile, tablet, desktop)

## Adding a New Company Page

1. Create a new folder (e.g. `company-name/`)
2. Add an `index.html` inside it
3. Push — it'll be live at `/company-name/`
